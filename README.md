# Python Exercises & Mini Projects

This repository contains small Python programs and exercises I built while learning Python fundamentals.
Each script focuses on practicing a specific concept such as logic, loops, functions, or libraries.

## Projects Included
- Alarm Clock – A command-line alarm using Python and pygame
- Guessing Game – A number guessing game with user interaction
- Hangman Game – Text-based hangman game
- Message Encryption – Simple message encryption script
- Bank Simulation – Basic banking logic practice

## Technologies
- Python
- pygame

## Purpose
This repository is for learning and practice.  
More structured and larger projects will be added separately (Hopefully :) ).
