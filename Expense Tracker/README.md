# Expense Tracker

A simple Python command-line program to track personal expenses. Users can add expenses, view all expenses, see a summary of their spending, and export data to a CSV file.

## Features

- **Add Expense:** Enter the name, amount, category, and date of an expense.  
- **Show Expenses:** View all recorded expenses in a readable format.  
- **Summary:** See total spending.  
- **Export Data:** Save all expenses to a CSV file (`expenses.csv`) for later use.  
- **Input Validation:** Handles invalid entries gracefully.

## Requirements

- Python 3.x
- Standard libraries: `datetime`, `csv`

No additional libraries are needed.
