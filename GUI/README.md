# PyQt5 Training Experiments

This repository contains multiple small experiments and exercises with **PyQt5**, the Python library for creating GUI applications.  
It is meant as a learning playground to understand widgets, layouts, styling, and interactions in PyQt5.

---

## Files Overview

- **ButtonsExperimenting.py** – Testing QPushButton functionality and signals.  
- **LabelExperimenting.py** – Playing with QLabel for text and images.  
- **LayoutLabels.py** – Experimenting with layout management for labels.  
- **LineEdit.py** – Working with QLineEdit input fields.  
- **RadioButtons.py** – Using QRadioButton widgets and handling selection.  
- **Styling.py** – Applying stylesheets to widgets for custom designs.  
- **checkBoxExperimenting.py** – Learning QCheckBox behavior and state handling.  
- **checkbox.py** – Additional checkbox experimentation.  
- **simpleIcon.py** – Displaying icons and integrating them into buttons or labels.  
- **onion.png** – Example image used in the icon or label experiments.

---

## Features Practiced

- Basic **widgets**: buttons, labels, checkboxes, radio buttons, line edits  
- **Layouts**: vertical, horizontal, and grid layouts  
- **Signals & Slots**: connecting user interactions to functionality  
- **Styling**: using Qt stylesheets to customize appearance  
- **Icons and images**: integrating PNG images into the GUI
