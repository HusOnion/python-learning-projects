# Simple Python To-Do List

A basic command-line To-Do List app written in Python.  
This project demonstrates the use of **lists, loops, functions, and input validation**.

---

## Features

- Show all tasks in the list  
- Add new tasks  
- Remove existing tasks  
- Handles invalid input gracefully  

> Note: Tasks are not saved after closing the program. Persistence will be added in future updates :).
